# Bubble Game
[![Ask DeepWiki](https://devin.ai/assets/askdeepwiki.png)](https://deepwiki.com/manishsuthar01/bubble_game)

A fun and fast-paced number-matching game built with HTML, CSS, and vanilla JavaScript. Test your speed and accuracy by popping the correct bubbles before the timer runs out!

## üéÆ How to Play

1.  **Start the Game**: Click the **Start** button to begin. The game area will fill with bubbles, each containing a random number.
2.  **Match the Number**: Look at the **Hit** value at the top of the screen. Your goal is to click on a bubble that matches this number.
3.  **Score Points**: For every correct bubble you click, your **Score** increases by 10 points. A new set of bubbles and a new **Hit** value will be generated.
4.  **Beat the Clock**: You have 60 seconds to score as many points as possible. The **Timer** at the top will count down from 60.
5.  **Game Over**: When the timer reaches 0, the game ends, and your final score is displayed. The **Start** button will be re-enabled for you to play again.

## ‚ú® Features

*   **Dynamic Bubble Generation**: A new set of bubbles with random numbers is created at the start and after every correct hit.
*   **Countdown Timer**: A 60-second timer adds a thrilling challenge to the game.
*   **Scoring System**: Earn 10 points for each correct match.
*   **Randomized Hit Target**: The target number changes after each successful hit, keeping the game engaging.
*   **Simple & Clean UI**: An intuitive interface built with plain CSS for a smooth user experience.
